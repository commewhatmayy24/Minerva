# Minerva

Minerva is an open-source atmospheric 3D psychological exploration game focused on immersive storytelling, liminal environments, and puzzle-driven gameplay.

## Features
- Cinematic environments
- Exploration-based gameplay
- Psychological horror elements
- Puzzle interactions
- Modular game systems

## Tech Stack
- Unity / Unreal Engine
- C#
- Blender
- GitHub

## Goals
The project aims to help beginner contributors learn game development through open-source collaboration.

## Contribution Guide
Contributors can help with:
- UI systems
- Environment design
- Character mechanics
- Optimization
- Documentation

## Setup
1. Clone the repository
2. Open in Unity/Unreal
3. Install dependencies
4. Run the project
